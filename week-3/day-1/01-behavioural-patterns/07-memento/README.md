# Memento

## Summary

a behavioral design pattern that lets you save and restore the previous state of an object without revealing the details of its implementation.

## Motivation

- Originator
- Memento
- Caretaker

## Example
- SaveGame

## Use Cases
- SaveGame (or other Serialization)
- Replays
- Undo/Redo
